# 🗓️ Ahmed Sayed Calendar - Python Project

## 📌 About The Project
This project is a **simple Calendar application built with Python** using the Tkinter graphical interface.  
It allows the user to choose a **year and month** and instantly display the full calendar for that month.

The program is designed to be **simple, fast, and easy to use**, making it a great example for beginners who want to learn how to build graphical applications in Python.

---

## ⚙️ Features
- Select **Year and Month**
- Display the **calendar instantly**
- Simple **Graphical User Interface**
- Built using **Tkinter**
- Beginner-friendly Python project

---

## 🖥️ Technologies Used
- Python  
- Tkinter GUI Library  
- Calendar Module  

---

## 🚀 How It Works
1. Run the Python file.
2. Choose the **Year**.
3. Choose the **Month**.
4. Click **GO**.
5. The calendar for the selected month will appear.

---

## 📷 Program Interface
A simple window that allows the user to select the year and month and display the calendar in a clean interface.
